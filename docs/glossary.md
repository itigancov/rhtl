# Timeline Glossary

- Timeline: The public product name for the React component primitive in this repository.
- Timeline primitive: A composable set of React components for chronological, process, roadmap, activity, milestone, and editorial layouts.
- Source-owned copy/paste usage: A distribution model where users copy the component source into their app and own future changes locally.
- Position: The `Timeline position` value that controls rail placement and text alignment.
- Rail: The visual vertical axis that connects markers and separators.
- Marker: The visual point for an item. Consumers can use the default dot or provide custom `TimelineMarker` children.
- Separator: The vertical line segment rendered between item headers and content.
- Content: The user-owned body rendered inside `TimelineContent`.
- Slot: A stable rendered part identified by `data-slot` for styling, debugging, and testing.
- Demo app: The local Vite app used to inspect practical Timeline compositions.
- Public release: The first user-facing release posture for Timeline as early but usable copy/paste source.
